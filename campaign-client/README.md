## Installation

For install all dependency:

```sh
npm install
```

## Create and configure .env.local file

For auth0 authentication need domain and clientId

```sh
VITE_DOMAIN=..........
VITE_CLIENTID=..........
```

For API

```sh
VITE_SERVER_API=http://localhost:5000
```

## Run and build

For run the project:

```sh
npm run dev
```

For build the project:

```sh
npm run build
```
